This application allows you to connect to the Hicrobot camera and scan its buses and their values. The table allows you to change the camera's bus configuration in real time. For each parameter, you'll see a list of available values, generated by polling the camera's SDK. For each input bus, you can run a listener that will perform specified actions when triggered. In this case, I used sending a message to the TCP port. For each output bus, you can also assign a listener that will wait for a message on the specified TCP port. If the incoming command matches the one you specified, an action will be performed—in this case, switching (by blinking the LED). To add your own processing methods, simply write your own class implementing the IActionMethod or IActionMethod interface functionality. All classes of these interfaces are queried via reflection, and your method will automatically appear in the drop-down list.
